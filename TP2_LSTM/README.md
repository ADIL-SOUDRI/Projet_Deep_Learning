# TP2 — Prédiction de la Consommation Électrique avec Stacked LSTM

## Description
Ce projet de Deep Learning a pour objectif de prédire la consommation électrique d’un foyer à l’aide d’un modèle Stacked LSTM (Long Short-Term Memory).

Le projet utilise des données de séries temporelles afin d’apprendre les variations de consommation électrique et de réaliser des prédictions automatiques.
